# encode
Simple URL Encoder & Decoder

### Requirements
* Node.js
* npm

### Usage
**install**
```bash
npm install -g TakutoYoshikai/encode
```


**encode string**
```bash
encode "http://localhost:8000/hello?q=20) = 1 OR 1=1 (1%2B1"
```

**decode string**
```bash
decode "http://localhost:8000/hello?q=20)%20=%201%20OR%201=1%20(1%252B1"
```
